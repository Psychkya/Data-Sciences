To determine k, we will plot graph to find elbow

#find elbow
kval <- 2:14
set.seed(1)
WSS <- sapply(kval, function(k){kmeans(iris.znorm, centers = k)$tot.withinss})
plot(kval,WSS,type = "l", xlab = "number of k", ylab = "within sum of squares")

Interesting point around k=5. the ss value actually increases for a bit after k=5. We will try again with min-max normalization to see if it makes any difference

normalize <- function(x){ return((x - min(x))/(max(x) - min(x)))}
iris.znorm2 <- as.data.frame(lapply(iris.z, normalize))
